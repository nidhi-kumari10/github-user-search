# 🔍 GitHub User Search

A sleek and modern web application to search and explore GitHub users in real-time. Easily discover developers, view their profiles, and sort them based on key metrics.

🌐 **Live Demo:** https://nd-githubprofile.netlify.app/

---

## ✨ Features

* 🔎 Search GitHub users by username
* 📊 Display user stats (followers, repositories, etc.)
* 🔀 Sort users by:

  * Followers
  * Repositories
  * Name
* 🧩 Toggle between **Grid** and **List** view
* ⚡ Fast and responsive UI
* 🎨 Clean and modern design

---

## 🚀 Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript**
* **GitHub REST API**

---

## 📂 Project Structure

```
├── index.html     # Main HTML file
├── style.css      # Styling
├── script.js      # Application logic & API calls
```

---

## ⚙️ How It Works

1. Enter a GitHub username in the search bar
2. The app fetches data using the GitHub API
3. Results are displayed instantly
4. Use sorting and view toggle for better exploration

---

## 🛠️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/nidhi-kumari10/github-user-search.git
```

2. Navigate to the project folder:

```bash
cd github-user-search
```

3. Open `index.html` in your browser
